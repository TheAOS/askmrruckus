# askmrruckus
Parody of Ask Mr. Robot

## Prerequisites
- node.js
- mongodb

## Installation
Clone the git repo:

    git clone https://github.com/TheAOS/askmrruckus.git
    
Install dependencies:

    npm install

Copy example config:

    cp config.json.example config.json

Then edit the config with your desired values. Then launch with

    node index.js
